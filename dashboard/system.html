Reward Function Design
R = 0.4×latency + 0.3×reliability + 0.2×throughput + 0.1×fairness
Latency (0.4)1.0 − util on chosen path
Reliability (0.3)1.0 − packet loss on path
Throughput (0.2)Normalized bytes transferred
Fairness (0.1)1.0 − |util_A − util_B|
Priority ×5Emergency/actuator multiplier
Drop Penalty−1.0 (unless congested: +0.1)
