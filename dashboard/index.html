What it does

Four pieces working together — an agent, a network, a brain to compare against, and a window to watch it all.

Deep RL Agent

A Dueling LSTM-DQN with Double-Q learning, a replay buffer and ε-greedy exploration. It reads a 26-feature view of the network and picks one of six routing paths to maximise a shaped reward.

Real SDN Fabric

A Ryu OpenFlow controller installs flow rules on Open vSwitch switches inside Mininet — a 3-cluster, 7-switch IoT topology carrying sensor, video, bulk and emergency traffic.

Baseline Showdown

Every decision runs a shadow classical policy — shortest-path, ECMP, round-robin, least-utilised — so you can see exactly where, and by how much, the AI wins on each metric.

Simulation Lab

Pick a topology and a scenario, then watch the real model route live traffic and train as you go. Every animated packet is the routing decision the data is actually made of.

How the agent thinks

Each cycle turns raw switch counters into a decision — and learns from what happens next.

1

Observe

Link utilisation, flow counts, loss, jitter & priority become a 26-feature state vector.

2

Decide

The LSTM-DQN scores every path and selects the action with the highest expected reward.

3

Route

The controller pushes OpenFlow rules so the chosen path carries the flow end-to-end.

4

Reward

Latency, reliability, throughput & fairness combine into one shaped reward signal.

5

Learn

A gradient step nudges the network; ε decays and the policy sharpens over time.
